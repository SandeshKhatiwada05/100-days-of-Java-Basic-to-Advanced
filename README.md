# Java 100 Days Challenge

Welcome to the **Java 100 Days Challenge**!  
This repository features daily lessons and programs designed to take you from **Java fundamentals to advanced concepts**. Topics include core Java, OOP, Swing, JDBC, and more. Build a strong Java foundation—one day at a time.

---

## Challenge Goal

**Master Java in 100 days** through consistent practice. Start with the basics (data types, I/O), advance to OOP, GUI development, database integration, and real-world projects.  
This challenge is ideal for students and self-learners seeking stepwise programming mastery.

---

## Topics Covered

- **Days 1–12:** Basics – Data Types, Input, Conditionals, Loops, Methods, Recursion
- **Days 13–28:** OOP – Classes, Encapsulation, Constructors, Inheritance, Abstract Classes, Interfaces, Method Overloading & Overriding
- **Days 29–40:** Advanced Java – Access Modifiers, Static/Final, Exception Handling, Threads
- **Days 41–50:** Collections – ArrayList, LinkedList, Deque, HashMap, Date & Calendar
- **Days 51–73:** GUI – Java Frames, JavaFX, Swing Forms & Dialogs
- **Days 74–84:** Projects – Attendance Checker, Batch Processing, Remote Method Invocation
- **Days 85–100:** Advanced Components – Tabbed Pane, Mouse/Key Listeners, Tree Components, JDBC, JSP/Servlets, final JavaFX project

Daily exercises reinforce both theory and practical skills for **gradual progression**.
